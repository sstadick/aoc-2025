# Advent of Code 2025 in Mojo

Create a new day:

```
bash ./newday.sh day01
```

Then solve!


## Prereqs

- [`pixi`](https://pixi.sh/latest/) (latest, `pixi self-update`) 
- `mojo` = "=0.25.7" (Installed via pixi)

This template and repo makes use of [extramojo](https://github.com/ExtraMojo/ExtraMojo) for nice to have things that are not yet in the stdlib
